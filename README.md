# Data Analytics Project

## Overview
This project demonstrates an end-to-end Data Analytics workflow, covering data extraction, cleaning, analysis, database management, dashboard development, and business reporting. The project leverages Python, SQL, Power BI, and Gamma to transform raw data into actionable insights and visual reports.

## Dataset
The dataset used in this project contains structured business data that was analyzed to uncover trends, patterns, and key performance indicators (KPIs). The data was processed and prepared for analysis through extensive cleaning and transformation.

## Project Workflow

### 1. Data Loading
- Imported the dataset into Python.
- Examined dataset structure and data types.
- Validated data quality.

### 2. Exploratory Data Analysis (EDA)
- Performed descriptive statistical analysis.
- Identified trends and relationships.
- Created visualizations to understand data distribution.

### 3. Data Cleaning
- Handled missing values.
- Removed duplicate records.
- Corrected inconsistent formats.
- Converted columns to appropriate data types.
- Treated outliers where necessary.

### 4. SQL Analysis
- Loaded cleaned data into a relational database.
- Wrote and executed SQL queries for:
  - Data retrieval
  - Aggregation
  - Filtering
  - Joins
  - KPI calculations

### 5. Dashboard Development
- Connected Power BI to the processed data.
- Designed interactive dashboards.
- Created KPI cards, charts, and filters.
- Enabled dynamic business insights through visual analytics.

### 6. Reporting
- Summarized findings and recommendations.
- Developed a presentation using Gamma.
- Prepared reports for stakeholder review.

## How to Run

### Prerequisites
- Python 3.9 or above
- PostgreSQL / MySQL / SQL Server
- Power BI Desktop

### Installation

```bash
git clone <repository-url>
cd Data-Analytics-Project
pip install -r requirements.txt
```

### Run Analysis

```bash
jupyter notebook
```

Open the notebook in the `notebooks` folder and run all cells.

### Execute SQL Queries
1. Import the cleaned dataset into your database.
2. Open the SQL script from the `sql` folder.
3. Execute the queries using your preferred SQL client.

### View Dashboard
1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data source if required.
3. Explore the interactive dashboard.
